Exercise 1: Explore the Diversity in the Simplicity

Review a most simplified HTML5 page in various browsers and devices to get a feel for the baseline in a bunch of different places.

  1. Load and view the first version of the scene from The Tempest in browsers that you have with you. Desktop, any devices. You can also find this web page hosted externally at http://lyzadanger.com/d/101.
  2. Review screen shots of the same page rendered in various devices (below).
  3. Try out the Opera Mini simulator (load up the version at http://lyzadanger.com/d/101).

Pro Tips

  1. The Opera Mini Simulator is a great tool
  2. Screen shots from (all versions of) Android? Do this:
    1. Install the Android SDK
    2. Make sure to install platform-tools
    3. Connect the Android device in question to USB
    4. In the Android SDK directory, find ddms under tools and launch it (You can also run ddms from a Terminal window in most Android SDK installs).
    5. Find the connected device in the Dalvik Debug Monitor window and use Device -> Capture Screen... to do a screen grab. Here's a screenshot.
