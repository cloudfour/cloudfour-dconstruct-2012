Exercise 2: Getting our Baseline Going

Add some common, common denominator CSS and viewport normalization

  1. Edit tempest.html
    1. Add a viewport tag
    2. Add a <link> tag to reference styles.css
    3. Update the <title> tag
  2. Edit styles.css
    1. Add CSS reset rules from http://meyerweb.com/eric/tools/css/reset/
    2. Add a rule for the html element and add a font reset.
    3. Add a font-family property to the html rule, as well.
